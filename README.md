# AMICIS Website

Dit is de officiële AMICIS bedrijfswebsite, gebouwd met Next.js, Tailwind CSS, ShadCN UI en Lucide React Icons.

## 🔧 Installatie

```bash
npm install
npm run dev
```

## 🌐 Deploy

1. Push naar GitHub
2. Verbind GitHub met [https://vercel.com](https://vercel.com)
3. Klik op 'Deploy'

Logo staat in `/public/AMICIS LOGO.png`
